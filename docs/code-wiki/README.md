## 仓库定位

该项目是一个基于 `React 18 + TypeScript + Vite` 的前端单页原型，主题为“把 AI Coding 新手引导做成游戏第一关”。当前版本聚焦最小闭环：

1. 告诉用户为什么要先获得 API Key。
2. 引导用户前往 DeepSeek 平台获取 API Key。
3. 返回页面粘贴 API Key 并执行本地模拟验证。
4. 展示“第一关完成”的过渡结果，为下一关埋点。

## 推荐阅读顺序

1. 先阅读 `01-项目总览.md`，建立整体认知。
2. 再阅读 `02-架构设计.md`，理解状态流与边界。
3. 接着查看 `03-模块详解.md`，进入具体实现细节。
4. 需要本地运行或排障时，查看 `04-依赖与运行.md`。
5. 需要继续开发时，查看 `05-开发方向规划.md`。

## 当前结论摘要

- 当前仓库是纯前端原型，没有真实后端服务和联网校验逻辑。
- 业务核心集中在 `Home` 页面、`DialogPanel` 组件、`useSceneStore` 状态中心与 `useTypewriter` 动画 Hook。
- 路由只有 `/` 一个页面，业务流程使用 Zustand 控制步骤切换。
- 测试主要覆盖首条新手引导链路与“上一步”回退能力。
- 仓库内已有产品文档与技术文档，但部分内容早于当前实现，需要以源码为准理解现状。
